# CC 商城 优惠券适用范围规则

> 文档版本：v4.2  
> 生效日期：2026-05-10

## 一、优惠券分类

CC 商城共发行 **5 类优惠券**：

| 类型 | 简称 | 形式 |
|---|---|---|
| 满减券 | 满 ¥X 减 ¥Y | 订单满额抵扣 |
| 折扣券 | X 折 | 订单按比例折扣 |
| 品类券 | 仅 XX 品类可用 | 限定商品分类 |
| 新人券 | 新注册用户专享 | 首单优惠 |
| 会员券 | 等级会员专享 | 按等级发放 |

## 二、通用使用规则

- 优惠券一次只能使用一张（同类型不可叠加）
- 优惠券**不可兑现**、不可转赠
- 优惠券有效期内不使用，过期作废，不可退还
- 退货时，优惠券**优先抵扣已退商品金额**，不退现

## 三、不可使用优惠券的商品

- **秒杀商品**：标注"秒杀价"的商品
- **预售商品**：定金/尾款付款的预售订单
- **品牌特卖区**：明确标注"特卖价不可用券"
- **二手 / 福利品**：标注"福利价"的商品
- **虚拟商品**：电子卡券、会员充值

## 四、适用商品识别

用户在购物车页面，可见每件商品下方标注：
- 🎫 **可使用优惠券**
- 🚫 **不可使用优惠券**

结算时系统会**自动剔除不可使用的商品**，仅对剩余金额抵扣。

## 五、特殊场景

### 5.1 多个商品订单
- 优惠券按"全部可用商品总价"计算是否满足门槛
- 抵扣金额按"可用商品价格占比"分摊

### 5.2 退货后优惠券处理
- **全单退货**：优惠券原路返回账户（仅限有效期内）
- **部分退货**：剩余订单仍满足门槛 → 优惠券保留；不满足 → 按比例追回抵扣额

### 5.3 优惠券获取途径
- 注册赠送（新人券 ¥10）
- 会员日发放（每月 8 号）
- 客服补偿（投诉处理后赠送）
- 签到积分兑换
- 营销活动领取
